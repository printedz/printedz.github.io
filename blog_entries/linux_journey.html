My GNU/Linux journey - 02/04/2023 (2nd of April of 2023. Yes, I like that format)

I've been distro hopping for a long time, however I think I've settled on OpenSUSE Tumbleweed. I really like how it's stable (I don't want my GRUB to break D:) yet is a rolling release distro with up-to-date packages. Here are some applications I use:
Web browser: SeaMonkey
E-mail client: SeaMonkey, I really like how well it integrates with the browser, and it has S/MIME support!
Instant messaging apps: Nheko, ArmCord, Konversations and Revolt
Desktop environment: KDE

I've found the KDE ecosystem to perfectly suit my needs, some would argue it's not polished and it's unstable but I haven't had any issues with it. In fact, I've had more problems with GNOME apps.
